# OC Solutions Costa Rica - Calculadora de Precio

Aplicacion web sencilla para calcular precios de venta en Costa Rica usando margen sobre venta, IVA y tipo de cambio.

## Que hace

- Calcula el precio de venta con la formula:

```text
precio de venta = precio en colones / (1 - porcentaje)
```

- Permite ingresar costos en CRC o USD.
- Convierte USD a CRC con tipo de cambio del BCCR.
- Permite usar tipo de cambio manual, BAC manual o BCCR.
- Calcula precio final con o sin IVA 13%.
- Muestra costo total, ganancia neta y margen real.
